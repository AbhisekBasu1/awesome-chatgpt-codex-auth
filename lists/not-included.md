# Not included

This list is for common submissions that should generally be rejected unless something materially changes.

## Generic API clients

Reject projects that only support `OPENAI_API_KEY` and do not document ChatGPT/Codex subscription sign-in.

## Account resale or sharing

Reject account marketplaces, shared ChatGPT account services, “cheap Plus” products, coupon farms, and services that sell access to someone else’s subscription.

## Quota bypass

Reject tools whose main purpose is bypassing rate limits, rotating accounts for abuse, or reselling subscription-backed capacity.

## Cookie/session-token scraping

Reject tools that require copying browser cookies or session tokens unless there is a compelling, well-documented, security-reviewed reason to list them as high-risk.
